Celestial Coordinate Systems

2020-11-16

This work is from part of UCB CS289A 2020 Fall Project S Final. See Github repo for more information.

A physical model is evaluated so that we can understand the problem more properly.

Solar-centered ecliptic coordinate system

Solar-centered ecliptic coordinate system is centered at the sun, using spring equinox as x+x+x+ or polar axis, ecliptic as xyxyxy plane.

Earth loacation (polar)

xe=(re,θe,0)\mathbf{x}_e = (r_e, \theta_e, 0)xe​=(re​,θe​,0)

where

θe=2πTeyt+θe0\theta_e = \frac{2 \pi}{T_{ey}} t + \theta_{e0}θe​=Tey​2π​t+θe0​

or (dirichlet)

xe=[recos⁡(2πTeyt+θe0)resin⁡(2πTeyt+θe0)0]\mathbf{x}_e = \begin{bmatrix} r_e \cos{(\cfrac{2 \pi}{T_{ey}} t + \theta_{e0})}\\ r_e \sin{(\cfrac{2 \pi}{T_{ey}} t + \theta_{e0})}\\ 0\\ \end{bmatrix}xe​=⎣⎢⎢⎢⎢⎢⎡​re​cos(Tey​2π​t+θe0​)re​sin(Tey​2π​t+θe0​)0​⎦⎥⎥⎥⎥⎥⎤​

other planets can have a similar definition.

Earth-centered ecliptic coordinate system

Solar-centered ecliptic coordinate system is centered at the earth, using spring equinox as x+x+x+ or polar axis, ecliptic as xyxyxy plane.

Equatorial coordinate system

Equatorial coordinate system is centered at the earth, using spring equinox as x+x+x+ or polar axis, equator as xyxyxy plane.

Planets location in such system is defined as right ascension α\alphaα and declination δ\deltaδ,

as (polar)

Xp=(Rp,π2−δ,α)\mathbf{X}_p = (R_p, \frac{\pi}{2} - \delta, \alpha)Xp​=(Rp​,2π​−δ,α)

as we normally do with spherical coordinate system (r,θ,ϕ)(r,\theta,\phi)(r,θ,ϕ)

or (dirichlet)

Xp=[Rpcos⁡δcos⁡αRpcos⁡δsin⁡αRpsin⁡δ]\mathbf{X}_p = \begin{bmatrix} R_p \cos\delta \cos\alpha\\ R_p \cos\delta \sin\alpha\\ R_p \sin\delta\\ \end{bmatrix}Xp​=⎣⎢⎡​Rp​cosδcosαRp​cosδsinαRp​sinδ​⎦⎥⎤​

Horizontal coordinate system

Horizontal coordinate system is centered at the observer, using local north as x+x+x+ or polar axis, local vertical up direction as z+z+z+.

Planets location in such system is defined as azimuth AAA and altitude aaa,

as (polar)

X^p=(Rp,π2−a,A)\hat \mathbf{X}_p = (R_p, \frac{\pi}{2} - a, A)X^p​=(Rp​,2π​−a,A)

as we normally do with spherical coordinate system (r,θ,ϕ)(r,\theta,\phi)(r,θ,ϕ)

or (dirichlet)

X^p=[Rpcos⁡acos⁡ARpcos⁡asin⁡ARpsin⁡a]\hat \mathbf{X}_p = \begin{bmatrix} R_p \cos a \cos A\\ R_p \cos a \sin A\\ R_p \sin a\\ \end{bmatrix}X^p​=⎣⎢⎡​Rp​cosacosARp​cosasinARp​sina​⎦⎥⎤​

Coordinate transformation

It is easy to transform between the solar-centered ecliptic coordinate system and the earth-centered ecliptic coordinate system. A planet with coordinate xp\mathbf{x}_pxp​ in solar-centered ecliptic coordinate system is at xp−xe\mathbf{x}_p - \mathbf{x}_exp​−xe​ in earth-centered ecliptic coordinate system.

xp−xe=[rpcos⁡(2πTpyt+θp0)−recos⁡(2πTeyt+θe0)rpsin⁡(2πTpyt+θp0)−resin⁡(2πTeyt+θe0)0]\mathbf{x}_p - \mathbf{x}_e = \begin{bmatrix} r_p \cos{(\cfrac{2 \pi}{T_{py}} t + \theta_{p0})} - r_e \cos{(\cfrac{2 \pi}{T_{ey}} t + \theta_{e0})}\\ r_p \sin{(\cfrac{2 \pi}{T_{py}} t + \theta_{p0})} - r_e \sin{(\cfrac{2 \pi}{T_{ey}} t + \theta_{e0})}\\ 0\\ \end{bmatrix}xp​−xe​=⎣⎢⎢⎢⎢⎢⎡​rp​cos(Tpy​2π​t+θp0​)−re​cos(Tey​2π​t+θe0​)rp​sin(Tpy​2π​t+θp0​)−re​sin(Tey​2π​t+θe0​)0​⎦⎥⎥⎥⎥⎥⎤​

cos⁡A⋅cos⁡a=−cos⁡ϕ⋅sin⁡δ+sin⁡ϕ⋅cos⁡δ⋅cos⁡H\cos A\cdot \cos a=-\cos \phi \cdot \sin \delta +\sin \phi \cdot \cos \delta \cdot \cos HcosA⋅cosa=−cosϕ⋅sinδ+sinϕ⋅cosδ⋅cosH

sin⁡A⋅cos⁡a=cos⁡δ⋅sin⁡H\sin A\cdot \cos a=\cos \delta \cdot \sin HsinA⋅cosa=cosδ⋅sinH

sin⁡a=sin⁡ϕ⋅sin⁡δ+cos⁡ϕ⋅cos⁡δ⋅cos⁡H\sin a=\sin \phi \cdot \sin \delta +\cos \phi \cdot \cos \delta \cdot \cos Hsina=sinϕ⋅sinδ+cosϕ⋅cosδ⋅cosH

Note

  • Planets are actually in ellipse orbits instead of circle ones and zzz is not 000 to be precise, here we made some simplification just to show the complexity of the problem
  • A slow motion of Earth's axis, precession, causes a slow, continuous turning of the coordinate system westward about the poles of the ecliptic, completing one circuit in about 26,000 years.
